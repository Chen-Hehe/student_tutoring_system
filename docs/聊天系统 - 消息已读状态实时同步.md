# 🦞 聊天系统 - 消息已读状态实时同步

**实现日期：** 2026-04-21  
**模块：** 实时聊天系统  
**优先级：** P0

---

## 📋 功能描述

实现聊天消息的已读状态实时同步：
- ✅ 发送方可以看到每条消息的已读/未读状态
- ✅ 接收方打开聊天窗口时自动标记为已读
- ✅ 已读状态通过 WebSocket 实时推送给发送方
- ✅ 无需刷新页面即可看到状态变化

```
学生 A                              后端                              教师 B
  |                                  |                                  |
  |--------发送消息 (isRead=false)-->|                                  |
  |                                  |------WebSocket 推送------------->|
  |                                  |                                  | 收到新消息
  |                                  |                                  |

```

## 💻 技术实现

### 后端实现

#### 1. ChatRecordService.java

**核心方法：**

```java
/**
 * 标记消息为已读，并推送已读状态给发送者
 */
@Transactional(rollbackFor = Exception.class)
public void markAsRead(Long currentUserId, Long senderId) {
    log.info("ChatRecordService.markAsRead - 标记已读：currentUserId={}, senderId={}", 
             currentUserId, senderId);
    
    // 1. 查询未读消息
    LambdaQueryWrapper<ChatRecord> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(ChatRecord::getSenderId, senderId)
        .eq(ChatRecord::getReceiverId, currentUserId)
        .eq(ChatRecord::getIsRead, false)
        .eq(ChatRecord::getDeleted, 0);
    
    List<ChatRecord> records = chatRecordRepository.selectList(wrapper);
    if (records.isEmpty()) {
        log.debug("没有需要标记为已读的消息");
        return;
    }
    
    // 2. 批量更新为已读
    for (ChatRecord record : records) {
        record.setIsRead(true);
        chatRecordRepository.updateById(record);
    }
    
    log.info("ChatRecordService.markAsRead - 已标记 {} 条消息为已读", records.size());
    clearCache(currentUserId, senderId);
    
    // 3. 推送已读状态给发送者（WebSocket）
    pushReadStatusToSender(senderId, currentUserId, records.size());
}

/**
 * 推送已读状态给发送者
 */
private void pushReadStatusToSender(Long senderId, Long readerId, int unreadCount) {
    try {
        Map<String, Object> readStatus = Map.of(
            "type", "read",
            "readerId", readerId,
            "senderId", senderId,
            "unreadCount", 0,
            "timestamp", System.currentTimeMillis()
        );
        
        log.info("【推送已读状态】senderId={}, readerId={}, unreadCount={}", 
                 senderId, readerId, unreadCount);
        chatWebSocketHandler.sendToUser(senderId, readStatus);
    } catch (IOException e) {
        log.error("推送已读状态失败", e);
    }
}
```

#### 2. ChatWebSocketHandler.java

```java
/**
 * 发送已读状态更新给指定用户
 */
public void sendReadStatus(Long userId, Long readerId) throws IOException {
    Map<String, Object> readStatus = Map.of(
        "type", "read",
        "readerId", readerId,
        "timestamp", System.currentTimeMillis()
    );
    log.info("【发送已读状态】userId={}, readerId={}", userId, readerId);
    sendToUser(userId, readStatus);
}
```

```jsx
const renderMessage = (msg, index) => {
  const isSelf = msg.senderId === currentUser?.id
  const isRead = msg.isRead === true
  
  return (
    <div key={msg.messageId || index}>
      <div style={{ /* 消息气泡样式 */ }}>
        <div>{msg.message}</div>
        <div>
          {msg.timestamp}
          {isSelf && (
            <span>
              {isRead ? (
                <span style={{ color: '#1890ff' }} title="已读">✓</span>
              ) : (
                <span style={{ color: '#999' }} title="未读">⏳</span>
              )}
            </span>
          )}
        </div>
      </div>
    </div>
  )
}
```

##### 2. WebSocket 监听已读状态更新

```javascript
wsService.onMessage((data) => {
  // 处理已读状态更新
  if (data.type === 'read') {
    console.log('收到已读状态更新:', data)
    setMessages((prev) => prev.map(msg => 
      msg.senderId === currentUser.id && 
      msg.receiverId === data.readerId && 
      !msg.isRead
        ? { ...msg, isRead: true }
        : msg
    ))
    loadConversations()
    return
  }
  // ... 其他消息处理
})
```

##### 3. 切换对话时自动标记已读

```javascript
// 关键：当选中对话变化时，自动标记为已读
useEffect(() => {
  if (selectedConversation?.userId) {
    console.log('切换对话，标记为已读:', selectedConversation.userId)
    chatAPI.markAsRead(selectedConversation.userId)
  }
}, [selectedConversation?.userId])

const selectConversation = (conversation) => {
  setSelectedConversation(conversation)
  localStorage.setItem('selectedConversation', JSON.stringify(conversation))
  // 切换对话时自动标记为已读
  if (conversation.userId) {
    chatAPI.markAsRead(conversation.userId)
    loadChatHistory(conversation.userId)
  }
}
```

##### 4. 优化滚动逻辑（刷新后自动到底部）

```javascript
const messagesEndRef = useRef(null)
const isLoadingRef = useRef(false)

const scrollToBottom = () => {
  if (messagesEndRef.current && messages.length > 0 && !isLoadingRef.current) {
    setTimeout(() => {
      messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' })
    }, 100)
  }
}

useEffect(() => {
  scrollToBottom()
}, [messages])

const loadChatHistory = async (userId) => {
  isLoadingRef.current = true
  setLoading(true)
  try {
    const result = await chatAPI.getChatHistory(userId)
    setMessages(result.data || [])
    await chatAPI.markAsRead(userId)
    setTimeout(() => {
      isLoadingRef.current = false
      scrollToBottom()
    }, 200)
  } catch (error) {
    isLoadingRef.current = false
  } finally {
    setLoading(false)
  }
}
```
